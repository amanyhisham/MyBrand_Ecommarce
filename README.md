# 🛍️ E-Commerce Web App

A simple and modern e-commerce website built using **HTML, CSS, JavaScript, and Tailwind CSS**.
The project displays products dynamically from an API and provides a smooth user experience.

---

## ✨ Features

* 🔐 User Authentication (Login / Register باستخدام LocalStorage)
* 🛒 Dynamic Products from API
* ⭐ Product Rating System
* 🔎 Search Functionality
* ➕ Load More Products
* 💖 Wishlist Icon (UI)
* 🛍️ Add to Cart Button (UI)
* 📱 Responsive Design (Mobile & Desktop)

---

## 🧰 Technologies Used

* HTML5
* CSS3
* Tailwind CSS
* JavaScript (Vanilla JS)
* DummyJSON API

---

## 📦 APIs Used

* https://dummyjson.com/products/category/womens-dresses
* https://dummyjson.com/products/category/womens-shoes
* https://dummyjson.com/products/category/womens-bags
* https://dummyjson.com/products/category/womens-jewellery
* https://dummyjson.com/products/category/tops

---

## 🚀 How It Works

1. Fetch products from multiple APIs
2. Combine all products into one list
3. Display products in a grid layout
4. Allow users to search for products
5. Load more products on button click
6. can put element in wishlist,cart
